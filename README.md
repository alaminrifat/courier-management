### Installation
1. Install **Xampp**
2. Copy this repository to ~/xampp/htdocs/
3. Start **Xampp Control Panel**
4. Start **'Apache'** and **'MySQL'** in Xampp 
5. Open your <em>browser<em> and go - [localhost/couier-management](localhost/couier-management)
